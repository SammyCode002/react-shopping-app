# React Shopping App

A Single Page Application (SPA) built with React and React Router. Features Oregon State University themed styling.

## Features

- Single Page Application with React Router
- 3 pages: Home, Order, Registration
- Dynamic order table with quantity controls (0-10)
- Registration form with validation
- Oregon State themed design (orange and black)

## Tech Stack

- React 18
- React Router DOM
- React Icons
- Vite

## Pages

| Page | Description |
|------|-------------|
| Home | Welcome page with app description |
| Order | Table displaying items with +/- quantity controls |
| Registration | Form to register for promotions |

## Run Locally
```bash
npm install
npm run dev
```

Visit `http://localhost:5173`
